Smart Street Light Management System using Solar Panels IoT

The project aims to improve energy efficiency and urban safety by using sensor-equipped street lights. These can automatically adjust their brightness based on ambient light levels and the presence of pedestrians and vehicles. We used ESP32 microcontroller for this project. The data collected by the sensors is sent to the microcontroller.

Hand Script Numeral Detection using ResNet

I developed an OCR system using OpenCV and CNNs. In preprocessing, images are converted to grayscale, thresholded, and contours identify characters. A pre-trained ResNet CNN model accurately recognizes and extracts characters, providing a list of recognized characters as output. This hybrid approach combines traditional computer vision techniques with deep learning.
